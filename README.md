# autoBooker
TimeEdit autoBooker

## Installation

Use the package manager python3 -m pip install pyautogui

```bash
pip install foobar
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
