# autoBooker
TimeEdit autoBooker script (is all you need, and life is easy.)

## Installation
```bash
python3 -m pip install pyautogui
python3 -m pip install pynput
```

## Usage
Put preferred rooms in preferredRooms.csv, one room per line.
```csv
C3 090
C3 091
C3 092
```
